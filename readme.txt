== Description ==
Introducing our TUMO International WordPress theme crafted specifically for global / international websites. This theme is exclusive and cannot be utilized on any other websites

== Installation ==
1. Upload the theme files to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance>Themes' screen in WordPress.
